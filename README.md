# NVIM Config Files

This repo contains the config files for my local NeoVim setup.

### How to configure

After you install [NeoVim](https://neovim.io/). Clone this repo using
```
git clone https://github.com/nipeshkc7/nvim-config ~/.config/nvim/
```

### Plugins used

* GitGutter
* CoC
* NERD-Tree
* CTrl-P
* CS

### ColorScheme

* GruvBox
